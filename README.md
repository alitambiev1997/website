# English Tutor Website

A modern, responsive, and bilingual (English/Czech) landing page for an English tutor.

## Features

- **Bilingual Support**: Instant switching between English and Czech.
- **Responsive Design**: Looks great on mobile, tablet, and desktop.
- **Modern UI**: Clean, blue-themed design with smooth animations.
- **Calendly Integration**: Embedded scheduling for free calls.
- **Contact Form**: Ready for integration with EmailJS.
- **SEO Optimized**: Meta tags and semantic HTML for better search visibility.

## Deployment to GitHub Pages

1.  **Create a GitHub Repository**:
    - Go to GitHub and create a new public repository (e.g., `english-tutor-site`).

2.  **Upload Files**:
    - Upload `index.html`, `style.css`, `script.js`, and the `assets` folder to the repository.

3.  **Enable GitHub Pages**:
    - Go to the repository **Settings**.
    - Click on **Pages** in the left sidebar.
    - Under **Source**, select `Deploy from a branch`.
    - Select `main` (or `master`) branch and `/ (root)` folder.
    - Click **Save**.

4.  **Visit Your Site**:
    - After a minute or two, your site will be live at `https://yourusername.github.io/english-tutor-site/`.

## Customization

-   **Images**: Replace the placeholder images in the `assets` folder (or update the CSS/HTML to point to real images).
-   **Calendly**: Update the `data-url` in `index.html` with your actual Calendly link.
-   **Contact Form**: To make the form functional, sign up for [EmailJS](https://www.emailjs.com/) and update `script.js` with your Service ID and Template ID.
